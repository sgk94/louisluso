Key Data Flows

1. Product Catalog
Zoho Inventory → Next.js API route → ISR cached pages (revalidate every ~15 min)
Public sees SRP. Logged-in stores see bespoke pricing from Clerk metadata or CRM lookup.
2. B2B Quote/Order
Store builds cart → submits order → API route creates Sales Order in Zoho Books → Ken reviews → converts to Invoice with payment link → store pays via Stripe
3. Dealer Locator
Zoho CRM (active dealer contacts with addresses) → API route → map with geolocation search
"Contact This Dealer" sends email with product info + customer contact details.
4. Become a Partner
Form submission → API route creates Lead in Zoho CRM → Ken reviews → approves → Clerk account created → welcome email with login
5. Inventory Sync
Zoho Inventory is source of truth. Website caches product data via ISR. Stock changes in Zoho reflect on site within ~15 min. No more WooCommerce stock scripts.
