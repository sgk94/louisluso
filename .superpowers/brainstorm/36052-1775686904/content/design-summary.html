Key Flows

B2B Order:
Login → browse catalog (bespoke pricing) → add to cart → review quote → submit → Sales Order in Zoho Books → Ken reviews → converts to Invoice with payment link → store pays via Stripe
Contact Dealer:
Customer on product page → "Find Nearest Dealer" → map search → "Contact This Dealer" → form (name, email, message) → email sent to dealer with product info + customer details → CC to admin@ and cs@louisluso.com
Become a Partner:
Form → Lead in Zoho CRM → Ken reviews → approves → Clerk account created → welcome email with login credentials
Product Data:
Zoho Inventory (source of truth) → Next.js ISR cached pages (~15 min revalidation) → public sees SRP, B2B sees bespoke pricing

Data Sources

Products, variants, stock, images → Zoho Inventory API
Invoices, sales orders, payments → Zoho Books API
Dealers, partner applications, contacts → Zoho CRM API
B2B auth, sessions, bespoke pricing tier → Clerk
Payment processing → Stripe (via Zoho Books payment links)
